# Taurify

### Technology Stack
- Frontend: Vue + TypeScript
- Desktop Framework: Tauri 2.0
- State Management: Pinia
- Build Tool: Vite

### Development Setup
#### Prerequisites
- Node.js >= 18.12 
- Rust (latest stable)
- pnpm (package manager)

```bash
# Install pnpm
npm install -g pnpm

# Install dependencies
pnpm install

# Start development server
pnpm tauri dev
```

### Production Build
```bash
pnpm tauri build
```
